这是个人创作的一个简易学校学生教师信息后台管理系统，所有数据均由mockjs生成
# 技术栈
使用了Vue2 + Element-ui + Axios + Vue-router 
# 功能列表
登录   √
学生信息增删改查（包括学生信息详情）   √
教师信息查看删除   √
班级周课程表查看   √
招生情况地区对比饼状图   √
